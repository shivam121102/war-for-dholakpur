# War for Dholakpur

War for Dholakpur is a fun web-based project inspired by the fictional world of **Dholakpur** from the famous cartoon universe.

The project presents a small adventure/battle concept where players interact with the game environment and experience a simple gameplay built using web technologies.

## About the Project

Dholakpur is a fictional kingdom from the popular Indian animated series **Chhota Bheem**.  
This project is inspired by that universe and represents a creative attempt to build a simple interactive web experience.

## Features

- Simple interactive gameplay
- Fun battle/adventure theme
- Inspired by cartoon-style storytelling
- Lightweight and fast web project

## Tech Stack

- HTML
- CSS
- JavaScript

## Purpose

The goal of this project is to practice **web development concepts** while building a fun and creative project.

It demonstrates:
- Frontend development
- Basic game logic
- Interactive web design
